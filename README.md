# my_gyazo
Alternative simple gyazo script for Ubuntu

Install
-------------------

```bash
  $ sudo ./install.sh
```

### Usage
#### From terminal
```bash
  [vigo@ubuntu ~]$ my_gyazo
```

#### From launcher
- You can drag `My Gyazo` file to launcher and click to run it

### Note
- Tested on Ubuntu 14.04
- This is a simple script, use at your own risk :D
